# Pratica3-SD

### Introdução 

Uma aplicação gRPC que funciona como um CRUD para gerenciamento de imóveis de uma
imobiliária. Cada imóvel é composto por:

● Título
● Endereço
● Tipo (casa/apto)
● Número de quartos
● Número de garagens
● Área construída
● Área do terreno
● Preço
